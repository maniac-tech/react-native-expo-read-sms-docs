# Installation

Install using npm:

```
npm i @maniac-tech/react-native-expo-read-sms
```

:::warning Development Build Required
This library uses native Android APIs and will **not** work with [Expo Go](https://expo.dev/go). You must use a [development build](https://docs.expo.dev/develop/development-builds/introduction/) or a bare React Native project.
:::
